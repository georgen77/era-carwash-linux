### ПРАВКА 3 — Голос: одна дата = дата заезда (checkin_date)

Найди обработчик голосовой команды создания кода замка.

Логика при одной дате в команде:
1. Дата = `checkin_date`
2. Найти бронь: `apartment` + `checkin_date`
3. Взять `checkout_date` из брони
4. Время по умолчанию: заезд 15:00, выезд 11:00

Пример: `"сделай код Сальвадор на 15 апреля"`
```
→ apartment=salvador, checkin_date=2026-04-15
→ находим бронь → checkout_date=2026-04-18
→ код: 15:00 15.04 — 11:00 18.04
```

### ПРАВКА 4 — Сообщение гостю: формат XXXX# с объяснением

Найди `notifyLockCode` (или аналог). Сообщение гостю — ТОЧНО такой формат:

```
Your access code for room ERA Deluxe near the beach is
XXXX#
(The # key is the UNLOCKING key on the keypad)
It will be valid during this period: From 15:00 29-03-2026 To 11:00 01-04-2026
```

- `#` вплотную к коду, без пробела
- Даты: `DD-MM-YYYY`, время: `HH:MM`
- Только английский язык

### БАГ #2 — Ошибка 500 при отправке грязного белья (enum)

**Симптом:**
```
Request failed with status code 500
{"error":"Invalid input value for enum item_type: \"Простыни\""}
```

**Причина:** Supabase ждёт английский ключ, бот шлёт русское название.

Найди запрос `create_dirty_linen_record`. В поле `item_type` должен уходить `key`, не `name`:

```javascript
// НЕВЕРНО:
{ item_type: item.name }  // "Простыни" — ошибка

// ВЕРНО:
{ item_type: item.key }   // "sheets" — OK
```

Маппинг:
```
sheets / duvet_covers / pillowcases / large_towels / small_towels
kitchen_towels / rugs / beach_mat / mattress_pad
```
